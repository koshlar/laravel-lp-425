# Микро проект Laravel для УП 425

## Как запустить

1) Установите пакеты

```bash
composer install
```

2) Скопируйте и переименуйте файл `.env.example` в `.env`
3) Укажите свои данные для подключения к БД

```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ВАШЕ_НАЗВАНИЕ_БД
DB_USERNAME=root
DB_PASSWORD=ВАШ_ПАРОЛЬ
```
